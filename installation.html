Installation

This guide provides step-by-step instructions for installing uframe.

Prerequisites

Before installing uframe, ensure you meet the following requirements:

  • Python >= 3.7

Ensure you have Python and pip (Python’s package installer) installed by running:

python --version
pip --version

Installation Steps

Option 1: Install via pip

The recommended way to install uframe is through pip. Run the following command:

pip install uframe

Option 2: Install from Source

To install from the source, first, clone the repository:

git clone https://github.com/URWI2/uframe.git
cd uframe

Then install it using:

pip install .

If you encounter any problems during the installation process, please create an issue.
